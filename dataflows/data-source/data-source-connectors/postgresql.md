# PostgreSQL

## Description

Use data on PostgreSQL as Data Source of CDP 365 by choosing PostgreSQL Connector.

## Authentication

Authentication information for PostgreSQL Connector includes:&#x20;

* Host Name or IP
* Port
* Service Name
* Username
* Password
